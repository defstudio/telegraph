Webhook позволяет Telegram доставлять updates в ваше Laravel-приложение, чтобы бот мог отвечать на команды из чатов и нажатия кнопок. Telegram принимает только публичный `https` URL, поэтому в local development нужен local tunnel вроде `ngrok`, Cloudflare Tunnel, Expose или другого аналога.

## Local development через tunnel

1. Запустите приложение Laravel:

```shell
php artisan serve
```

2. Откройте публичный `https` tunnel к тому же порту. Для стандартного `php artisan serve` это обычно порт `8000`:

```shell
ngrok http 8000
```

3. Скопируйте публичный `https` URL tunnel и добавьте его в `.env`. Можно задать весь `APP_URL`, либо только webhook domain для Telegraph:

```dotenv
APP_URL=https://example.ngrok-free.app
TELEGRAM_WEBHOOK_DOMAIN=https://example.ngrok-free.app
TELEGRAPH_WEBHOOK_SECRET=your-secret-token
```

Не коммитьте реальные bot token и webhook secret. Для production используйте secret manager или защищённые переменные окружения.

4. Если приложение использует cached config, сбросьте cache перед регистрацией webhook:

```shell
php artisan config:clear
```

5. Зарегистрируйте webhook. Аргумент `{bot_id}` обязателен, если в базе больше одного bot:

```shell
php artisan telegraph:set-webhook {bot_id} --secret="${TELEGRAPH_WEBHOOK_SECRET}"
```

Telegraph зарегистрирует route вида `https://example.ngrok-free.app/telegraph/{token}/webhook`, если `TELEGRAPH_WEBHOOK_URL` не переопределён. Если tunnel перезапущен и URL изменился, обновите `.env`, выполните `php artisan config:clear` и запустите `telegraph:set-webhook` снова.

## Проверка регистрации

Проверьте, какой URL сейчас знает Telegram:

```shell
php artisan telegraph:debug-webhook {bot_id}
```

В local troubleshooting можно временно включить подробный dump входящих webhook updates:

```dotenv
TELEGRAPH_WEBHOOK_DEBUG=true
```

Не оставляйте подробный webhook debug включённым в production без необходимости: payloads могут содержать пользовательские данные.

## Programmatic setup

Webhook можно зарегистрировать программно через метод `registerWebhook()` у bot model:

```php
/** @var TelegraphBot $bot */
$bot->registerWebhook(
    secretToken: config('telegraph.webhook.secret'),
)->send();
```

Сквозной local development flow с HTTPS tunnel, webhook secret и handler для `/start` описан в [Build your first Telegram bot with Telegraph](6.build-your-first-bot).

Для параметров `dropPendingUpdates`, `maxConnections` и `secretToken` смотрите [webhook registration reference](../15.webhooks/2.registering-webhooks).
