Associating one or more chats to a bot, it is enabled to send messages to that chat and interacting with commands

> [!NOTE]
> Чтобы получить _chat_id_, отправьте команду `/chatid` в чат после [настройки webhook](3.setting-webhook) для bot. Если вы тестируете локально, проверьте, что tunnel URL всё ещё активен и совпадает с URL из `php artisan telegraph:debug-webhook {bot_id}`.


### through an artisan command

```shell
php artisan telegraph:new-chat {bot_id}
```

the bot_id argument is mandatory if you have created more than one bot

Если вы настраиваете первый bot от установки до первого сообщения, используйте [Build your first Telegram bot with Telegraph](6.build-your-first-bot).

### programmatically

If you are implementing a custom bot creation logic, you can create a new chat from a `TelegraphBot` model:

```php
use DefStudio\Telegraph\Models\TelegraphBot;

$bot = TelegraphBot::query()->firstOrFail();

$chat = $bot->chats()->create([
    'chat_id' => '123456789',
    'name' => 'Personal chat',
]);
```
